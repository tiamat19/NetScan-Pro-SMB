# NetScan Pro SMB

## Overview
NetScan Pro SMB is a local network security assessment tool for small businesses 
and IT consultants. It scans your local network, discovers every connected device, 
detects known CVE vulnerabilities, checks NIS2 compliance, and generates a 
professional PDF report in under 5 minutes.

No cloud. No SaaS. Runs entirely locally on Windows.

## Key Features
- Device discovery via ARP with vendor detection
- Device role identification with confidence scoring
- Integrated CVE database (EternalBlue, BlueKeep, OpenSSH RCE, and more)
- Per-device Security Score with A-F rating
- Automatic NIS2 / ACN 379907-2025 compliance mapping
- Professional PDF report with non-technical executive summary
- Prioritized remediation plan
- Interactive network topology map
- Baseline comparison between scans
- Asset inventory with CSV export
- Scheduled automated scans with email alerts
- White-label branding (custom logo on reports)
- Bilingual interface (Italian/English)

## Target
Small businesses and IT consultants managing SMB networks, 
especially Italian companies subject to NIS2 compliance requirements.

## Requirements
- Windows 10/11 (64-bit)
- Administrator privileges
- Local network connection
